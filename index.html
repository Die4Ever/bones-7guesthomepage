I don't know how old this website is, but it doesn't exist anymore. Luckily the Wayback Machine captured most of it (missing a bunch of images). Unfortunately the Wayback Machine is a little slow, so navigating around the house in this way is a bit of a pain, and the split screen isn't sized properly for modern screens.
So I made this tweaked version that can be easily self-hosted or downloaded, and it runs much faster.
